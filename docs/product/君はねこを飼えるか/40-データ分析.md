## 2. 「遺棄・拾得（45%）」の推定

### 2.1 問題の所在

「遺棄・拾得」という分類は、以下の2つのケースを含む可能性があります：

- **A. 元々野良猫を拾得したケース** → 飼い主の判断ミスとは無関係
- **B. 飼い主が何らかの理由で捨てたケース** → ゲーム設計で防ぐべき対象

**問題**: 統計データだけでは、A と B の内訳が不明。

### 2.2 推定の方針

以下の情報源から、「遺棄・拾得」に含まれる可能性のある理由を推定します：

1. **明示的な理由の合計を計算**
2. **後悔理由の調査結果との照合**
3. **論理的推定による内訳の仮説構築**

### 2.3 明示的な理由の合計

猫の保護依頼理由のうち、明示的な理由の合計：

```
飼い主の病気・傷病等: 24%
転居: 8%
動物の老齢・傷病等: 7%
苦情: 6%
経済的理由: 4%
離婚: 1%
問題行動: 1%
──────────────
合計: 51%
```

残り：
```
遺棄・拾得: 45%
不明: 4%
──────────────
合計: 49%
```

### 2.4 「後悔理由」調査との照合

複数の記事から抽出した、飼い主が実際に後悔した理由：

#### 調査1: NPO法人ねこほーむ

**手放しに直結する理由**として挙げられたもの：

- 旅行・出張が多く世話できない
- 経済的に継続困難（→ 4%として既に計上）
- アレルギー発症
- 生涯面倒を見る自信喪失
- 保護猫の定期報告がストレス

#### 調査2: au損害保険

**47%の飼い主が「大変だった、後悔したと感じたことがある」**

内訳：
- 生活面：26%（引っ越し、夜間の睡眠中断）
- お金：20%（医療費、フード代）
- お世話：20%（適切な世話の難しさ）
- 猫の生態：16%（夜間の鳴き声、予期しない行動）

#### 調査3: 日本ペットフード

後悔する12の理由の中で、飼育放棄につながりやすいもの：

- 医療費の負担（長期化・高額化）
- 生活の自由が失われる
- 外出の制限

### 2.5 「遺棄・拾得（45%）」の推定内訳（根拠付き）

以下の仮説を立てます：

| 推定される理由 | 推定割合 | 根拠 |
|---|---|---|
| **元々野良猫を拾得** | **20～25%** | - 出典論文で「野良猫・犬を拾ったが飼育困難」が定義として明示<br>- 「遺棄・拾得」という項目名自体が「拾得」を含む<br>- 日本の猫の保護経路として「野良猫を拾った」ケースが一般的 |
| **計画外の繁殖** | **5～10%** | - 複数の後悔理由記事で「不妊手術の重要性」が言及される<br>- 不妊手術未実施で子猫が産まれ、全頭飼えないケース<br>- 一般的な猫の飼育放棄理由として知られる |
| **アレルギー発症** | **3～5%** | - NPO法人ねこほーむの記事で「手放しに直結する理由」として明示<br>- 事前検査しても飼い始めてから発症する場合がある |
| **生活の変化（結婚・出産等）** | **3～5%** | - NPO法人ねこほーむで「生涯面倒を見る自信喪失」が手放しに直結すると明示<br>- 「家族の状況変化」は後悔理由で頻出<br>- 離婚（1%）とは別の家族構成の変化 |
| **多頭飼育崩壊** | **2～3%** | - 計画外の繁殖と関連<br>- 一般的な保護団体の報告で頻出する問題 |
| **ペット不可物件への転居** | **2～3%** | - 「転居（8%）」とは別に、ペット可物件が見つからなかったケース<br>- ベネッセ調査で「猫可物件が少なく住居探しに苦労」と明示 |
| **近隣トラブル（苦情の一部）** | **3～5%** | - 全動物で「苦情（6.04%）」が明示的に存在<br>- 出典論文で「近隣や第三者からの苦情が理由」と定義<br>- 鳴き声、臭い、脱走などによる近隣トラブル |
| **その他・不明** | **2～7%** | - 出典論文で「不明（4%）」が存在<br>- 上記に分類できない個別の理由 |

#### 推定の前提と制約

**前提：**
- 「元々野良猫を拾得」のケースは、飼い主の事前判断では防げない
- ただし、「拾った後に飼えなくなった理由」は防げる可能性がある

**制約：**
- この推定は複数の調査結果と論理的推論に基づくが、直接的な統計データは存在しない
- 「遺棄・拾得」という分類の定義が曖昧なため、正確な内訳は不明

## 3. データからの考察

### 3.1 後悔理由と放棄理由のギャップ

**重要な発見**: 後悔理由の調査では「47%が大変だった、後悔した」とあるが、実際の放棄に至るケースは一部である。

| 項目 | 後悔理由での頻度 | 放棄理由での割合 | ギャップ |
|---|---|---|---|
| 睡眠への影響（夜間の鳴き声等） | 頻出 | 1%（問題行動） | 大 |
| 衛生面（臭い・汚れ） | 頻出 | 6%（苦情）の一部？ | 中 |
| 家具・物への被害 | 頻出 | 1%（問題行動） | 大 |
| 生活の自由度への影響 | 頻出（26%が生活面で大変） | 明示的な項目なし | 大 |
| 経済的負担（医療費等） | 頻出（20%がお金で大変） | 4%（経済的理由） | 中 |

#### 考察

1. **後悔理由と放棄理由の構成が異なる**
   - 後悔理由で頻出する「睡眠への影響」「衛生面」「家具被害」は、放棄理由では1%（問題行動）と非常に少ない
   - この差は以下のいずれかの可能性がある：
     - **可能性A**: 多くの飼い主は大変だと感じながらも、責任を持って飼育を継続している
     - **可能性B**: 後悔理由が放棄理由として正直に申告されていない（社会的に言いにくい、または「遺棄・拾得」に含まれている）
   - **重要**: 可能性Bがある限り、後悔理由も飼育放棄の潜在的リスクとして扱うべき

2. **放棄理由で明示されているのは「不可抗力」が多い**
   - 飼い主の病気・傷病等（24%）：飼い主自身の事情
   - 転居（8%）：住環境の変化
   - 動物の老齢・傷病等（7%）：猫自身の事情
   - これらは社会的に説明しやすい理由であり、統計に現れやすい

3. **「遺棄・拾得（45%）」の内訳が鍵**
   - この中に「言いにくい理由」が含まれている可能性が高い
   - 推定では20～25%が「元々野良猫を拾得」（ゲーム対象外）
   - 残りの20～25%には、後悔理由（睡眠、衛生、経済的負担等）が含まれている可能性がある

### 3.2 経済的理由の二面性

出典論文では「経済的理由」が4.70%と明示されているが、後悔理由の調査では「お金：20%」が大変だった理由として挙げられている。

これは以下の2つに分離できる可能性：

1. **基礎生活費（事前に分かる）**
   - 毎月のフード代、トイレの砂代
   - 生涯126万円という固定費
   - 事前に計算すれば判断できる

2. **想定外の医療費（事前に分からない）**
   - 病気治療が長期化・高額化
   - 「お金：20%」の主要因と推定
   - 後悔理由では頻出するが、放棄理由では4%

→ 後者（想定外の医療費）は「遺棄・拾得」に含まれている可能性もある

### 3.3 問題行動の位置づけ

- **放棄理由としては1%と非常に少ない**
- **後悔理由としては頻出**（鳴き声、破壊行動）
- **考察**: 問題行動は「捨てる」までは至らないが、「後悔する」原因としては重要

## 4. まとめ：データ分析から得られた知見

### 4.1 統計データの確定事実

1. **遺棄・拾得（45%）が最大の理由** - ただし内訳不明
2. **飼い主の病気・傷病等（24%）が2番目** - 依然として重要
3. **問題行動（1%）は想定より少ない** - 放棄理由としては軽微
4. **経済的理由（4%）は明示的に存在** - 想定外の医療費が主因と推定

### 4.2 推定に基づく知見

1. **遺棄・拾得の約半数（20～25%）は「元々野良猫を拾得」と推定** - ゲーム設計の対象外
2. **残りの半数（20～25%）は防げる可能性がある**:
   - 計画外の繁殖（5～10%）
   - アレルギー発症（3～5%）
   - 生活の変化（3～5%）
   - 近隣トラブル（3～5%）
   - その他

### 4.3 「捨てられる原因」と「後悔する原因」の整理

#### 後悔する主な原因（調査データより）

- 日常のお世話の継続性（20%）

- 衛生面の現実（頻出）

→ これらは「放棄」には至らないが、「後悔」につながる

#### 既に解決済み（調べれば済む）の要素

以下は、後悔理由・放棄理由として**ほとんど挙がらない**要素です。事前に調べて判断できるため、ゲームで扱う優先度は低いと判断されます。

##### 1. 経済的負担（基礎生活費）

- 毎月のフード代、トイレの砂代
- 生涯126万円という固定費
- 後悔理由「お金：20%」に含まれるが、大半は想定外の医療費
- 基礎生活費は事前に計算すれば判断できる

##### 2. 時間的余裕（基礎世話時間）

- 朝夕最低30分、トイレ掃除、ブラッシング
- 後悔理由「お世話：20%」に含まれるが、大半は継続の難しさ
- 基礎的な時間は事前に見積もれば判断できる

##### 3. 住環境の制約

- ペット不可物件、部屋が狭すぎる
- 法的リスクが周知されており、事前に確認すれば判断できる
- 後悔理由・放棄理由としてほとんど挙がらない（「転居：8%」とは異なる）

##### 4. 必要なグッズと準備

- 初期費用50,000〜150,000円、必須アイテムのリスト
- 多くのWebサイトで情報が整備されており、事前に調べれば準備できる
- 後悔理由・放棄理由としてほとんど挙がらない

##### ゲーム設計への示唆

- これらの要素は、チェックリストや基本情報として提示する程度でよい
- 疑似体験で扱う優先度は低い
